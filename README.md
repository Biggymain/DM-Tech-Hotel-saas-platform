# DM-Tech-Hotel-saas-platform
An API-first, multi-tenant Enterprise Property Management System (PMS) and Hospitality ERP. Designed for unified management of hotels, F&amp;B outlets, and nightlife venues with integrated IoT door access and real-time inventory logic.

# HotelSaaS Platform (Enterprise Edition)

A comprehensive, cloud-native Hospitality Management System (HMS) designed to bridge the gap between core property management and modern guest experiences. 

### 🏨 Core Modules
- **Unified PMS:** Multi-property management with advanced room inventory and guest folio logic.
- **F&B Engine:** Integrated Point-of-Sale (POS) logic for Restaurants, Bars, and Lounges.
- **Nightlife & Events:** Specialized module for Club/Lounge guest lists, VIP table bookings, and bottle service.
- **IoT Gateway:** Abstracted hardware layer for seamless integration with RFID and BLE card access systems (ZKTeco, Orbita, etc.).
- **Automated GRC:** Built-in compliance logging and audit trails for financial and operational security.

### 🛠 Tech Stack
- **Framework:** Laravel 12 (PHP 8.3+)
- **Architecture:** Domain-Driven Design (DDD) with a Multi-Tenant Database Strategy.
- **Real-time:** Integrated WebSocket support for live occupancy and inventory updates.
- **API:** RESTful API documentation for 3rd-party integrations (Channel Managers, Door Locks).
